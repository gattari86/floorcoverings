# Floor Coverings International Houston Heights - Diagnostic Report

A professionally designed, mobile-friendly diagnostic report for Floor Coverings International Houston Heights, commissioned by Poppy Marketing and Consulting.

## Overview

This report provides a comprehensive forensic analysis of the digital and operational standing of FCI Houston Heights, examining:

- **40% Web Traffic Decline** analysis (September-November 2025)
- **Entity Audit** of owner Reinel Solano and franchise constraints
- **Competitive Landscape** analysis (Roberts Carpet, Greater Houston Flooring, Lowe's)
- **Technical & Content Audit** of website and GBP optimization
- **5-Phase Strategic Recovery Roadmap** for market dominance

## Deployment

### Vercel (Recommended)

1. Push this repository to GitHub:
```bash
git init
git add .
git commit -m "Initial commit: FCI Diagnostic Report"
git remote add origin https://github.com/gattari86/floorcoverings.git
git branch -M main
git push -u origin main
```

2. Connect to Vercel:
```bash
npm install -g vercel
vercel
```

3. Select your GitHub repository and deploy. The site will be live in seconds.

### Local Testing

Run a local server to preview the report:

```bash
# Using Python
python3 -m http.server 3000

# Using Node (if installed)
npx http-server

# Then visit: http://localhost:3000
```

### Three Root Causes of Traffic Decline

| Cause | Impact | Solution |
|-------|--------|----------|
| **Sept 2025 Bot Filter** (30%) | Ghost data removal | Recalibrate baseline |
| **October/November Core Updates** (40%) | Algorithm shifted to "popularity" | Increase review velocity |
| **I-610 Construction** (30%) | Physical location friction | Market mobile model as advantage |

## Customization

To customize the report:

1. **Colors**: Edit the CSS variables in `<style>` section:
   ```css
   :root {
       --primary: #2c5f8d;     /* Main blue */
       --secondary: #d4795f;   /* Rust/terracotta */
       --accent: #f5a623;      /* Gold */
   }
   ```

2. **Content**: Edit the HTML sections directly. The structure is semantic and easy to modify.

3. **Contact Info**: Update the CTA button with your email:
   ```html
   <a href="mailto:your-email@example.com">Get Started Today</a>
   ```
